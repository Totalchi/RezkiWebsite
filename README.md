# RM Bygg & Montage AB - Website

## How to open in IntelliJ IDEA

1. Open IntelliJ IDEA
2. Click **File > Open**
3. Select this folder (`rm-bygg-website`)
4. Right-click `index.html` in the project panel
5. Click **Open In > Browser** to preview

## Files
- `index.html` — the complete website (single file, self-contained)

## Features
- Dark / Light mode toggle
- EN / SV language toggle
- Booking form (3 tabs: meeting, quote, invoice)
- Animated hero with 3D canvas
- Gallery, testimonials, process steps
- Fully responsive (mobile + desktop)
- Logo embedded (no external image files needed)
