How We Scope Work

Three common project setups

Solar for a detached home

Typical scope: survey, production estimate and full installation plan

For solar work we start with roof orientation, shading, current electricity use and available roof area. The result is a written recommendation with a proposed system size, expected production range, installation method and what needs to happen with the network operator before commissioning.

Usually for
Detached homes and small properties
Includes
Site survey, quote, planning and grid paperwork
On-site time
Often 1-2 days for a villa
Optional add-ons
Battery storage and EV charging
Documents
Written scope before ordering
Shared EV charging

Typical scope: load balancing, cable routing and clear user setup

For a shared charging setup we review fuse capacity, parking layout, cable routes and how usage should be measured. The goal is to size the system correctly from the start so the customer understands cost drivers, expansion options and who remains responsible for what after handover.

Usually for
Housing associations, small offices and mixed-use properties
Focus
Load balancing and future-proof cable capacity
Planning
Board material and quote breakdown if needed
On-site work
Depends on charger count and distribution board changes
Handover
Inspection report and user guidance
Roof renovation

Typical scope: condition check first, then the right level of repair or replacement

Roof work starts with condition rather than assumptions. We look at moisture, underlay, flashings, drainage and structural health before recommending a partial repair or a full replacement. That keeps the quote tied to the actual condition of the roof rather than a sales target.

Usually for
Detached homes, townhouses and small apartment buildings
Review first
Moisture, timber condition, drainage and access
Can include
Tiles, metal, felt, gutters and facade work
Guarantee
Written workmanship coverage in the quote
Optional coordination
Scaffolding, solar preparation and follow-up fixes
